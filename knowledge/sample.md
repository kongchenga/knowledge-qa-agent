# Python 编程语言简介

> Tags: python, 编程, 教程

> Created: 2026-06-01T10:00:00

---

Python 是一种广泛使用的高级编程语言，由 Guido van Rossum 于 1991 年首次发布。它以其简洁、易读的语法而闻名，非常适合初学者和专业开发者。

## 主要特性

1. **简单易学**：Python 的语法清晰直观，接近自然语言。
2. **解释型语言**：Python 代码逐行执行，无需编译。
3. **动态类型**：变量不需要声明类型。
4. **丰富的标准库**：Python 拥有庞大的标准库，覆盖文件 I/O、网络编程、数据科学等。
5. **跨平台**：可在 Windows、macOS、Linux 等系统上运行。

## 应用领域

- Web 开发（Django、Flask）
- 数据科学和机器学习（NumPy、Pandas、Scikit-learn）
- 自动化脚本
- 人工智能和深度学习（TensorFlow、PyTorch）
- 桌面应用（PyQt、Tkinter）

## 示例代码

```python
def greet(name):
    return f"你好, {name}!"

print(greet("世界"))
```

Python 的哲学强调代码的可读性和简洁性，被亲切地称为"可执行的伪代码"。
